Comment jouer au jeu ?

Pour commencer à jouer, téléchargez simplement notre application depuis notre site officiel. Aucune installation complexe ni configuration n’est requise ! Suivez simplement les instructions ci-dessous en fonction de votre système d’exploitation.

  • Pour Windows : Après le téléchargement, ouvrez un terminal et executez la commande suivante : reg add HKEY_CURRENT_USER\Console /v VirtualTerminalLevel /t REG_DWORD /d 0x00000001 /f. Puis localisez le fichier et double-cliquez dessus pour démarrer l'application. Le jeu se lancera automatiquement sans étapes supplémentaires.

  • Pour Linux : Ouvrez un terminal dans le dossier où le fichier a été téléchargé, puis tapez les commandes suivantes pour rendre le fichier exécutable et lancer le jeu : chmod +x ./Yams.exe && ./Yams.exe

Une fois ces étapes réalisées, vous pouvez commencer à jouer immédiatement. Préparez-vous pour des parties palpitantes et suivez vos scores grâce à notre application !
